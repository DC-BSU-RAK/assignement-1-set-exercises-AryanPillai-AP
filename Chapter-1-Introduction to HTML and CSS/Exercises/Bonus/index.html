HTML & CSS Quick Reference Guide

A super simple guide for common HTML tags and CSS properties.

HTML Tags

<p> - Paragraph

Used to define a block of text.

<p>This is a paragraph.</p>

<a> - Anchor

Creates hyperlinks to other pages or sites.

<a href="https://example.com">Visit Site</a>

<img> - Image

Embeds an image into the page.

<img src="image.jpg" alt="Description" />

<div> - Division

A generic container used for grouping elements.

<div>Content here</div>

<span> - Inline Container

Used to style small inline portions of text.

<span style="color:red;">Red text</span>

<ul> & <li> - Unordered List

Creates a bullet-point list.

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

<h1> to <h6> - Headings

Defines headings of different levels.

<h1>Main Title</h1>
<h2>Subtitle</h2>

<code> - Inline Code

Displays code snippets inline with text.

<p>Use the <code>let</code> keyword in JavaScript.</p>

<pre> - Preformatted Text

Displays text exactly as written (preserves whitespace).

<pre>
Line 1
  Line 2
    Line 3
</pre>

<button> - Button

Defines a clickable button.

<button>Click Me</button>

CSS Properties

color

p {
  color: #333;
}

background-color

body {
  background-color: #f4f4f4;
}

font-family

h1 {
  font-family: Arial, sans-serif;
}

padding

section {
  padding: 20px;
}

margin

article {
  margin-bottom: 30px;
}

border

pre {
  border: 1px solid #ccc;
}

text-align

header {
  text-align: center;
}

box-shadow

.code-box {
  box-shadow: 0 0 10px rgba(0,0,0,0.1);
}
